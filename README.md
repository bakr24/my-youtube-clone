# YouTube Clone (HTML/CSS)

A responsive YouTube-style front-end layout built using HTML and CSS. This project focuses on demonstrating mastery of the CSS Box Model, Flexbox, and CSS Grid.

## Features
- **Responsive Layout:** Automatically adjusts for desktop and mobile screens.
- **Search Header:** Persistent top navigation bar.
- **Sidebar:** Navigation menu for site sections.
- **Video Grid:** Dynamic grid-based layout for video thumbnails.

## Technologies Used
- HTML5
- CSS3 (Flexbox & CSS Grid)
